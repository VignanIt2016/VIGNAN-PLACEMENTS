Selection procedure

This is basic test the test is exactly same as there is for normal TCS Drive.

TCS visits the following colleges –

Cream colleges (Top 40 colleges of India according to TCS)
Premium colleges (Top 100 colleges of India according to TCS)
Grade A,B,C colleges (Top 1000 colleges of India)
While TCS Digital only visits the following – 

Cream colleges
Premium colleges
Grade A
For Cream, Premium colleges the test sections are – 

English Proficiency
Analytical & Logical Aptitude
Programming Logic
Coding / Advanced Coding
For Grade A colleges the test is divided into two sections –

Test 1

email Writing
Quantitative Aptitude
TCS Technical MCQ Questions
Coding Test
Test 2

English Proficiency
Analytical & Logical Aptitude
Programming Logic
Coding / Advanced Coding
